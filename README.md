# UML-LLAMA-LoRA Dataset

## Overview

This repository contains a dataset of 6,884 requirement–PlantUML pairs 
used for training a Large Language Model (LLaMA 3.2) with LoRA fine-tuning.

## Dataset Format

Each example is structured as:

{
  "input": "Textual requirement",
  "output": "PlantUML code"
}

## Contents

- Dataset
- LoRA adapter weights
- Inference code
- Evaluation scripts
